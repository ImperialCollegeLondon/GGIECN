# GGIECN
paper submitted

## Prerequisites

- [Anaconda](https://www.anaconda.com/download/)
- [PyTorch](https://pytorch.org)

```
conda install pytorch torchvision cudatoolkit=11.0 -c pytorch
```

- opencv

```
pip install opencv-python
```


## Usage

### Train and predict

```
python GGIECN.py
optional parameter:
--data_path                  
--backbone_type               
--crop_h                  
--crop_w                     
--batch_size                  
--epochs                   
--save_path          
--model_weight
--predict_save_path
```
